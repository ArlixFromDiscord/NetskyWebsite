About NC Network

NC Network is a Discord Server with technology and anime in mind.

Founded on July 14th, 2017 after a failed attempt at merging Crashie's Lair and Netsky Network, NC Network s for Netsky-Crashie Network.

After the seperation it stayed as NC Network. It was re-merged on October 11th, 2017.

NC Network was merged with Aurora Community on October 25th, 2017 and has been growing steadily.

NC Network iand s the hub for all things relating to The Crashie Brand, and AeroFusion. The Crashie Partners also reside there.

What is the NC Network ShareX uploader?

The NC Network ShareX uploader is a continuation of the aurorame.me ShareX uploader made by Jake#0009.

That sounds good. Can I join it?

You sure can! There are a a few things you need to do first. Firstly, you must be in NC Network. You can join from the link above. Secondly, you need to read the rules/restrictions. You can read them here. After that, you can apply for a key here.

I have joined. How do I set it up?

Download the ShareX template here, open it in a text editor (such as Notepad++) and change INSERT_KEY_HERE to your key. Then just import it into ShareX.
